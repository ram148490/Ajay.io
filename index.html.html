Artifact 01 · Personal Finance · Interactive Dashboard
Monthly Budget Dashboard
Introduction

A fully interactive personal finance dashboard enabling users to track income, analyze spending, monitor savings goals, and navigate month-by-month data — built without external charting libraries.

Objective
  • Apply BI dashboard design principles to a real-world personal finance scenario.
  • Translate raw financial data into actionable visual insights using custom SVG charts.
  • Demonstrate end-to-end data storytelling through KPIs, trends, and monthly recommendations.
Process
  • Defined KPIs: total income, expenses, net savings, and savings rate with month-over-month change.
  • Built bar, donut, and trend line charts from scratch using SVG with live interactivity.
  • Added a data entry panel supporting manual input and CSV file upload with real-time chart updates.
